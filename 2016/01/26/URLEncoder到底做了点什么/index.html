URLEncoder到底做了点什么

JDK中的描述

Utility class for HTML form encoding. This class contains static methods for converting a String to the application/x-www-form-urlencoded MIME format.

转换规则

1、a-z , A-Z,0-9 不用变  
2、特殊字符 ".","_","*","-" 不用变  
3、" " 空格转换成"+"  
4、其他的转换成 "%xy" 格式
